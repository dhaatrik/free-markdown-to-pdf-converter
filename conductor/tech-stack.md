# MarkPDF Tech Stack

## Language
- **TypeScript**: Used for robust, type-safe application logic.

## Frontend Framework & Build Tool
- **React (v19)**: Core UI framework for component-based architecture.
- **Vite**: Fast development server and optimized production build tool.

## Styling
- **Tailwind CSS**: Utility-first CSS framework for rapid and responsive UI styling.

## Markdown Processing
- **React-Markdown**: For safely parsing and rendering Markdown.
- **Remark-GFM**: Adds support for GitHub Flavored Markdown (tables, strikethrough, task lists).
- **React-Syntax-Highlighter**: For code block highlighting.

## Testing
- **Vitest**: Blazing fast unit test framework.
- **React Testing Library**: For testing React components in a user-centric way.
